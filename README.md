# Commercial Vehicle Health Monitoring API

This project is a REST-based backend application built with Python Flask for monitoring the health condition of commercial vehicles. It stores vehicle and sensor data in SQL Server and evaluates key operating parameters such as engine temperature, oil pressure, battery voltage, and brake wear.

The application follows a clean layered structure using DTOs, models, services, repositories, and routes. It demonstrates backend development principles, REST API design, SQL Server integration, request validation, and basic rule-based diagnostic logic for commercial vehicle systems.
